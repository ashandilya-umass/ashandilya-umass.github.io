The code on the left defines a script that allows users to input a medical function or disease. 
                    It then checks if the input matches any of the diseases in a predefined list. 
                    If a match is found, it retrieves information about the disease from Wikipedia using web scraping with BeautifulSoup. 
                    It then displays the information in a tkinter window along with a clickable link to the Wikipedia page for further reading. 
                    The script also generates a SMILES code for a random combination of functional groups associated with the inputted disease. 
                    These functional groups were derived from the known structures of medical drugs used for those specific diseases and their SMILES codes.
                    Finally, it visualizes the molecular structure of the generated SMILES code using RDKit and matplotlib.
